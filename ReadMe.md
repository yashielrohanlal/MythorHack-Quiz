## App Description

The **Myth or Hack Quiz App** is an interactive Android application developed using Kotlin in Android Studio. The app helps users distinguish between real-life productivity hacks and common internet myths through a flashcard-style quiz.

Users are presented with various statements and must decide whether each is a **true life hack** or a **false myth**. The app provides instant feedback, tracks the user’s score, and allows them to review all answers at the end.

---

## Purpose of the App

The purpose of this app is to:

* Educate users about common misconceptions
* Improve critical thinking and decision-making
* Provide an engaging and interactive learning experience

## Features

* Welcome screen with app introduction
* Flashcard-style quiz interface
* True (Hack) / False (Myth) answer buttons
* Immediate feedback after each answer
* Automatic score tracking
* Final score screen with performance message
* Review screen showing all questions, answers, and explanations
* Smooth navigation between screens

## Project Structure

* `MainActivity` → Welcome screen
* `QuizActivity` → Handles quiz logic and questions
* `ScoreActivity` → Displays final score and feedback
* `ReviewActivity` → Shows all questions with correct answers

---

## How the App Works

1. The user starts on the welcome screen
2. Clicking **Start** opens the quiz
3. Questions are displayed one at a time using a loop
4. The user selects **Hack** or **Myth**
5. Feedback is shown immediately
6. The score updates automatically
7. After all questions, the score screen appears
8. The user can review all answers and explanations
